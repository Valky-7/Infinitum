# 🌐 Portal Institucional - Infinitum

Projeto desenvolvido com Angular e Tailwind CSS com o objetivo de simular um portal institucional de uma nação fictícia chamada **Infinitum**.

---

## 📌 Objetivo

Desenvolver uma aplicação **SPA (Single Page Application)** utilizando Angular Router para navegação entre páginas, apresentando informações institucionais do país Infinitum.

---

## 🚀 Tecnologias Utilizadas

* Angular
* Tailwind CSS
* TypeScript
* HTML5
* CSS3

---

## 🧩 Estrutura do Projeto

O sistema é dividido em componentes reutilizáveis e páginas principais:

### 🔹 Componentes

* Header (menu de navegação)
* Footer (rodapé)

### 🔹 Páginas

* **Home (/)** → Apresentação da nação
* **Perfil (/perfil)** → Informações detalhadas do país
* **Contato (/contato)** → Formulário de contato

---

## 📊 Sobre o País - Infinitum

**Características Iniciais:**
Economia digital baseada em dados e serviços em nuvem. Centro financeiro de criptoativos.

**Situação Atual:**
Uma tempestade solar severa danificou as redes globais, deixando o país offline e sem economia por semanas.

**Desafio:**
Desenvolver uma base econômica física para garantir a sobrevivência durante apagões digitais.

---

## 🎨 Funcionalidades

* Navegação por rotas (Angular Router)
* Layout responsivo com Tailwind CSS
* Componentização (Header e Footer fixos)
* Identidade visual própria (bandeira e favicon)
* Interface moderna com tema tecnológico
